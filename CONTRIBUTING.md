# Contributing

Thanks for your interest in Hermes Automation Examples.

This project shares automation workflows for creators building digital businesses.

## How You Can Contribute

You can contribute by:

- Improving workflow examples
- Adding new automation ideas
- Sharing integration patterns
- Improving documentation
- Reporting issues or suggestions

## Before Making Changes

For major changes, please open an issue first to discuss your idea.

## Areas We Welcome

Examples:

- Payhip automation
- Telegram bot workflows
- Make scenarios
- Notion database systems
- Creator business automation

## Pull Requests

When submitting a pull request:

- Explain what problem the change solves
- Describe the improvement
- Keep documentation clear and simple

Thank you for helping improve Hermes Automation Examples.
