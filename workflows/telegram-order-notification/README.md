# Telegram Order Notification Workflow

## Overview

Automatically notify creators when a customer purchases a digital product.


## Workflow

Payment received

↓

Webhook triggered

↓

Make automation

↓

Telegram notification sent

↓

Customer information stored in Notion


## Tools

- Payhip
- Make
- Telegram Bot
- Notion


## Use Cases

- Digital product sellers
- Template creators
- Online course creators
